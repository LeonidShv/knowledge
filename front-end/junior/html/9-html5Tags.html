HTML5 Tags

There are many tags in html5, (semantic, audio, video, canvas, and any more..)

embed - element can show audio, video, files (useful for shoving privacy on a website)

dialog - shows pop up above other element in the middle of screen by X coordinate

details - allow switch content, show or hidde block with content

base - you can use in head, can break all elements with src and href

in head - base href="https://website.daslab.app/", + , in body link with href="use-cases/#samplers", = , will equal https://website.daslab.app/use-cases/#samplers

map - add links on image

address - link on author

meter, progress - element with progressbar

noscript - if JS is disabled on the browser, you will see text inside this tag
