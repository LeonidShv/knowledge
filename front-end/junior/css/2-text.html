Responsive text

There are a few ways how to make responsive text.

For responsive text we can use:

1)viewport width (vw), font-size: 1vw;

2) media query,

3) formula from css-tricks, for html tag, then need to use rem for font-size

html: font-size: calc([minimum size] + ([maximum size] - [minimum size]) * ((100vw - [minimum viewport width]) / ([maximum viewport width] - [minimum viewport width])));

h1: 2rem;
