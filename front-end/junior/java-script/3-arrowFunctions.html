arrow functions

Arrow functions are special: they don't have their own this. So don't use them for object's methods. If we use this inside an arrow function, then its value is taken from outside

arrow functions are anonymous

don't use call, aply, bind

        const a = {b: 1};

        let arrow = () => {
            console.log(this.b)
        }

        arrow.call(a); // undefined

Can not be used as constructors.

Can not use yield, within its body.
