Array's functions

Change arr:

  • push (...items) – adds elements in the end
  • pop() – removes and return element from the end
  • unshift(...items) – adds elements in the start
  • shift() – removes and returns element from the start
  • splice(position, deleteCount, ...items) – starts from index 'position', remove amount 'deleteCount' elements and adds 'items'.
  • sort(func) - sorts and also returns new arr
  • reverse() - changes elements order, (also returns new arr)
  • arr.fill(value, start, end) - let a = []; a.length = 5; a.fill(1, 0, 5)
  • arr.copyWithin(target, start, end)

Don't change arr:

  • slice(start, end) - returns new arr, copy elements from 'start' to 'end' (not include 'end')
  • arr.concat(...items) - returns new array arr + items
  • indexOf/lastIndexOf(item, pos) - find item, pos - index for starting searching, it will return -1 if doesn't find anything
  • includes(value) - returns true/false
  • find/filter(func) - returns first or all values, when func gets true
  • findIndex(func) - the same as find, but return index instead of elem
  • forEach(func) - calls the function for each elem, doesn't return anything
  • map(func) - returns new arr, and use func for each elem
  • split/join - converts string to arr, and back
  • reduce(func, initial) - calculates a value from all array
  • Arra.isArray(arr) - return true/false
  • arr.some(fn) - it will return true if one fn(elem) return true
  • arr.every(fn) it will return true, if every fn(elem) return true
