Prototypes

__proto__, prototype - internal properties of object, function, array.

In primitives (number, str) are temporary object wrapper, which allows writing own methods in the prototype of primitives

String.prototype.show = function() { alert(this); };
arr.__proto__ === Array.prototype

__proto__ - can use for the class instance - ar1.__proto__

prototype - can use for class - Array.prototype

nill and undefined don't have temporary object-wrapers
