Semantic theory

Semantics responsible for - accessibility, search bots, code standardization

A semantic element clearly describes its meaning to both the browser and the developer. Examples of non-semantic elements: div, span - tells nothing about content.

Examples of semantic elements: form, table, header - Clearly defines its content.
